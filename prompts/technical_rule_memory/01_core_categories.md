# 核心分类记忆

## 缺流程要素

定义：
事项应具备的审批、决议、签字、比价、验收、台账、标准、依据文件等流程链条缺失或不完整。

高频触发特征：

- 无送货单
- 无签收记录
- 无实物图片
- 无发放标准
- 无两委会决议
- 无验收情况说明
- 无验收现场图片
- 缺审批资料
- 缺比价资料
- 缺会议决议
- 未留痕

典型判断模式：

- 事项本应有流程或依据，但缺了
- 流程存在，但关键节点无记录、无依据、无佐证
- 事前审批、事中留痕、事后验收任一环缺失

## 附件内缺少内容

定义：
凭证、考核表、说明、评估报告、验收材料等附件中缺少关键内容，导致证明力不足或真实性存疑。

高频触发特征：

- 无服务商签字
- 无图片
- 无现场资料
- 无细化条款
- 无情况说明
- 后附资料不完整
- 后附资料无法证明真实性

典型判断模式：

- 后附资料有，但缺关键内容
- 附件存在形式，但无法支撑结论或真实性

## 流程时间倒置

定义：
合同、审批、验收、支付、确认等流程的时间顺序颠倒，出现先执行后签约、先验收后确认、先支付后到期等情况。

高频触发特征：

- 合同签订日晚于服务期
- 合同签订日晚于租赁起始日
- 合同签订日晚于开工日
- 验收日期早于工程量确认单时间
- 支付时间早于到期时间
- 公司成立日晚于合同签订日

典型判断模式：

- 应先发生 A 再发生 B，但文本中出现 B 先于 A

## 长期挂账未处理

定义：
往来款、应收应付、专项应付款等账项长期挂账，超过合理期限仍未清理、转销或确认。

高频触发特征：

- 挂账 3 年以上
- 挂账 5 年以上
- 长期挂账未清理
- 长期未处理
- 长期应收款
- 长期应付款未清理

典型判断模式：

- 核心不是科目用错，而是“挂着不处理”

## 财务做账不规范

定义：
会计科目使用错误、确认时点错误、收支确认不规范、账务处理方式不合理、明细核算缺失等问题。

高频触发特征：

- 科目设置不规范
- 科目使用错误
- 未按权责发生制核算
- 直接冲减收入
- 未计提折旧
- 未确认收入
- 未确认支出
- 收支不匹配
- 未设置二级明细
- 串户

典型判断模式：

- 账应该怎么做，但现在没按会计逻辑做
- 不是流程没走，而是账务反映错了

统一规则：

- 优先统一输出“财务做账不规范”
- 不再优先新造“做账不规范”这种并行分类

## 合同内容存在问题

定义：
合同条款本身存在缺失、模糊、不一致、不合理，影响履约、结算或责任界定。

高频触发特征：

- 条款不清晰
- 未约定服务期限
- 未约定考核事项
- 未约定验收事项
- 签订方不一致
- 合同内容不严谨
- 合同约定不合理

典型判断模式：

- 问题主要出在合同文本本身

## 内容一致性问题

定义：
合同、台账、口径、考核表、服务商名称、人数、金额、图斑、实际情况之间不一致。

高频触发特征：

- 名称错误
- 与实际不一致
- 与会议纪要不一致
- 与合同约定不一致
- 台账与实际不一致
- 图斑信息不准确
- 明细、人数、金额差异过大

典型判断模式：

- 多个信息源互相对不上

## 业务高度重合

定义：
同类支出、服务、补贴、费用或工作内容重复设置、重复计价、重复支付、交叉覆盖。

高频触发特征：

- 重复支付
- 重复采购
- 重复列支
- 服务内容雷同
- 与原有工资/补贴重复
- 年度结算中已有，另行支付

典型判断模式：

- 不是单纯多付，而是与另一项业务或费用高度重合

## 违反相关规定

定义：
明确违反法规、制度、政策红线或普遍约束性规范。

高频触发特征：

- 违反八项规定
- 法律风险明显
- 明确违反规定
- 不应列支却列支

适用条件：

- 文本中存在较明确的规范性判断
- 如果只是“程序不规范”，不要轻易升格到此类

## 关联交易

定义：
与本村、本单位、关联人员、关联企业发生利益或交易联系，存在潜在利益输送或独立性不足风险。

高频触发特征：

- 法人与本村村民、干部、亲属相关
- 供应商、承包方与内部人员存在关联

适用条件：

- 文本应出现较明确的关联线索

## 其他问题

定义：
无法稳定归入上述主分类，或者问题虽然明确，但仍停留在异常现象层。

常见情况：

- 补偿标准不一致
- 税率差额未扣减
- 名目错误
- 异常但尚不足以判断为固定模式
